Freegle Doodles

Freegle Doodles are the Freegle Logos that have been altered to celebrate particular events and add a little more interest to the Freegle site.

We are taking our lead from Google Doodles http://www.google.com/doodles/about

Who can come up with Doodles?

Anyone can come up with a doodle, see the guidelines below. So please have a go, you will be credited for any doodles published. Drop me a line at graphics@ilovefreegle.org to let me know if you like to have a go and I'll create a list of doodlers.

What events do Freegle want doodles for?

As we are just trying this out we thought that we'd keep the list fairly modest in number. If we get a lots of people wanting to doodle for Freegle we can expand the list. I've started with a short list of days well known (I think ) to the UK public, it looks a bit heavy on the saints days but they tend to mark a day for that country to celebrate. However if you think other days should be on the list drop me a line at graphics@ilovefreegle.org.

Guidance on Producing a Freegle Doodle

What a Doodle should contain

A freegle doodle needs to have the word Freegle distinguishable within the design when its sized to 160 by 160 pixels. If possible it should try to contain an element of the Freegle heart in there too, although to what extent is down to the artist. Finally, it needs to reference the event.

Selection

If we get to the luxurious position of getting several designs in for a particular event, I'll email the group of contributors to get opinions. Should there need to be a casting vote or final decision the doodle organiser will do it (I don't mind being unpopular).

Format

This is where you can pick up the standard Logos in SVG format.

Once you create a Freegle Doodle send it to graphics@ilovefreegle.org again in SVG format so it can be resized for different uses.

Credits

Any Doodles used will be credited to the creator (although I'm not sure how we will show who that is ?)

Copyright

By creating a Doodle and sending it in for use you are giving Freegle the right to use it on its pages and websites without charge. If you want freegle to stop using the design just drop us an email and we will remove it from our pages and websites.

Old designs

Some old designs are stored here http://photobucket.com/freegledays

Geek guidance

First, decide on a filename that doesn't confict with existing doodle filenames. Next, make sure the doodle fits tightly within its bounds, ie no white space around it.

The small website logo needs to be 160x160px. Create a jpg or png file this size. Log in to perch and add a new ornament, uploading the image file. This goes to ilovefreegle/images/doodle on spock. A task every hour copies all the doodles over to web1. The create_daily_tips script is run overnight on spock and web1 - but can be run manually when required. If there is a doodle for today, the sitelogo and directlogo symlinks are updated to point to it; otherwise the standard ones are used.

Now upload various size versions of the doodle to 'shared' on web1 and spock. In perch create suitable shortlinks for each image uploaded.   Only use alphanumeric chars in the shortlink.

The Facebook profile photo should be 960x960 pixels and the cover photo 851x315 pixels - though Facebook will resize for you.

The Twitter main photo needs to be square and the header photo 1252x626 pixels.
