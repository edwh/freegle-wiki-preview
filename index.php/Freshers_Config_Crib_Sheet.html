This is a crib sheet for use alongside the standard-message config used on the training communities. The button names on other configs may differ, but the principle and use of the buttons will be much the same.

Automatic rejections

These send a message to the member and automatically reject the post - no further action required:

  • Animals - Reject
  • Duplicate Message - Reject
  • Medicines - Reject
  • Non Landfill Items - Reject
  • Charity event - Reject
  • Selling - Reject
  • Swapping - Reject
  • Vouchers - Reject
  • Weapons - Reject
  • Copyright - Reject

The last six will all send a message you can edit if needed, and then automatically reject the post - again, no further action required.

Blank letters

Only use these if none of the standard messages are appropriate.

  • Blank Letter - Leave: write a message to the member telling them what's wrong with their post and what you've done to correct it. After sending, you'll need to Approve the message.
  • Blank Letter - Reject: tell the member what they've done wrong; the post will be automatically rejected. No further action required.

Edits

For these, edit the post first, then send the note to tell the member what you've done. After sending, you'll need to Approve the edited message.

  • Borrowing - Edit: remove the borrowing aspect.
  • Cash offered for item - Edit: can be used for both Wanteds and Offers, editing to suit.
  • Charity event ad within a post - Edit: remove the charity event aspect.
  • Swapping - Edit: remove the swapping aspect.
  • Personal Details - Edit: remove the relevant details, then edit this message to suit the individual case.

Other

  • Edit Lower Case: changes the message from ALL CAPS to lower case automatically. After use, you'll need to Approve the message.
