This page shows the Freegle logos for use on websites or printed material. Choose the version that best suits your intended use, so you get the highest possible quality. There are also links to the original vector graphics files, which you can use to create your own graphics or print at any size without loss of quality. You may also want to look at Using the Logo.

There's also a page of website Badges with code ready to drop into a webpage or your group description.

Vector graphics

The above logos are also available in SVG format. You can use these in many drawing programs, including the free Inkscape. They can be printed at any size without loss of quality, so are ideal for large banners or posters.

Fully vector versions

Use these if you don't have the Mesmer font.

File:FreegleHeartMesmer vector.svg Freegle heart on a white background.

File:FreegleHeartMesmer trans outer vector.svg Freegle heart with transparent outer and white background for the heart. For use on most non-white backgrounds.

File:FreegleHeartMesmer trans vector.svg Freegle heart with transparent background. For use on backgrounds that contrast with Freegle green.

Versions with original text
