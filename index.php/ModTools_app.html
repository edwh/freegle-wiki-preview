Appearance
From Freegle Volunteer Wiki

The ModTools app for Android and iOS was first released in 2016. The app has the functionality of the https://modtools.org website with extra app features such as notifications when there is work to do.

ModTools app 0.4+

Note: Google and Facebook logins do not work.

This version of the ModTools app shows the live ModTools website within the app, as well as handling push notifications. So: the app is always up-to-date with the latest ModTools website changes.

Initial screen

On first use, the first screen below shows, assuming that you have enabled notifications. You can just press the Start ModTools button to show the normal ModTools screen. You will need to sign in to ModTools again.

However you will probably want to enable and connect notifications. Enabling notifications is done within an Android or iOS popup - but can also be turned on and off in the *phone* settings for the ModTools app.

To tell the Freegle server that you want notifications, enter your modding email and press Connect. If successful, the screen either shows "Success" or "Connected".

When you next start the app, it will automatically show the normal ModTools screen. However you can go back to the initial screen then you can close the ModTools window using the X in Android or Done in iOS.

You can manage notifications in the usual way within the phone settings for the app.

Using ModTools

When you press Start ModTools for the first time then you will have to sign into ModTools. Currently only a Freegle login and Yahoo login work. Once logged in you may use ModTools as normal.

If a work notification arrives while not on the app, then tapping on the notification will start the app and show the relevant ModTools screen. (If you are currently using the app, the screen will not change if you tap on the notification.)

If you wish, you can close the app in the normal way eg by swiping it away.

Note that opening external links within ModTools will not open the links in another page eg in Chrome or Safari. Instead they will replace the current screen; you can return by swiping back.
