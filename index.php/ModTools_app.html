The ModTools app for Android and iOS was first released in 2016. It has the functionality of the https://modtools.org website, plus extra app features such as notifications when there's work to do.

ModTools app 0.4+

Note: Google and Facebook logins do not work.

This version of the app shows the live ModTools website within the app, and handles push notifications - so the app is always up to date with the latest ModTools website changes.

Initial screen

On first use, assuming you've enabled notifications, the first screen shows. Press Start ModTools to go to the normal ModTools screen - you'll need to sign in to ModTools again.

You'll probably want to enable and connect notifications, too. Enabling notifications is done via an Android or iOS pop-up, but can also be turned on and off in the phone's settings for the ModTools app.

To tell the Freegle server you want notifications, enter your modding email and press Connect. If successful, the screen shows "Success" or "Connected".

Next time you start the app, it automatically shows the normal ModTools screen. You can still return to the initial screen and close the ModTools window using the X (Android) or Done (iOS).

You can manage notifications as usual in the phone's settings for the app.

Using ModTools

The first time you press Start ModTools, you'll need to sign in. Currently only a Freegle login and Yahoo login work. Once signed in, use ModTools as normal.

If a work notification arrives while you're not using the app, tapping it starts the app and shows the relevant ModTools screen. (If you're already using the app, tapping the notification won't change the screen.)

You can close the app as normal, e.g. by swiping it away.

Opening external links within ModTools won't open them in another app, such as Chrome or Safari - instead they replace the current screen; swipe back to return.
