Capturing screenshots

For information on capturing and editing screenshots, see:

  • PNGs for Screenshots
  • How to Take a Screen Shot (Screen Capture) (at WikiHow)

Everyone seems to have their favourite software for this. Some free options are:

  • Windows Paint - often already installed on machines running Windows
  • Irfanview - irfanview.com
  • Paint.net - filehippo.com/download_paint-net
  • GIMP - the-gimp.en.softonic.com (for the keen, as it's quite a big and complex app)
  • Gadwin - www.gadwin.com/download

For many of these, only the basic version is free; enhanced versions may only work for a one-month trial period.

Most have their own help files and settings. Some let you crop and adjust images before saving or emailing them.

For the wiki, it's best to save your screenshot to the desktop, or to a specific folder, ready to load into the wiki.

Uploading files

Freegle has its own section for uploads - see Help:Managing files.

In the toolbox on the left-hand side of any page you'll find 'Upload file'. This takes you to a special page: click Upload and search for the photo or screenshot on your computer. Give it a unique title, since there's usually only one screenshot per page. Your file then gets its own "File Description Page" within the File namespace of the wiki. You should edit this page to add more information about the file - for example, many public wiki projects care about copyright, so explain where you got the file from and what its copyright status is. Add any other useful notes there too.

Using images

Once you've uploaded a file, you can use it in an article with some special wiki syntax to create the link - see Help:Images. You can either link to the image or insert it directly. In edit mode, on the page where you want it to appear, place the cursor on the line and click the picture icon in the edit tools. Enter the page title (everything after "File:") and, when it shows in the preview window, choose to align it left, right or centre. You can then resize it on the page.
